How to use this form: Fill in every field marked REQUIRED before the app can go live. Fields marked OPTIONAL improve the experience but the app works without them. Return the completed form (or a filled copy) to the app team.

2 · Programme / Rundown

This is the most important section. Every session on every day needs to be listed. The app uses this to power the live "Happening now" strip, the "Up next" cards, and 1-hour-before reminders sent to delegates.

For each day, provide:

FieldDetails needed
Day label & date REQUIRED
e.g. "Day 1 — 14 September 2026"
Day theme / title OPTIONAL
e.g. "Arrival & Opening Ceremony"

For each session / item on that day, provide:

FieldDetails needed
Start time REQUIRED
24-hour format, Frankfurt local time — e.g. 09:00
Session title REQUIRED
e.g. "Opening Ceremony", "Keynote: The Future of Multilateralism"
Venue / room REQUIRED
Building name and room, e.g. "Main Hall, Westin Frankfurt"
Session type REQUIRED
One of: keynote · panel · workshop · visit · meal · social
Gather / assembly time OPTIONAL
If delegates must gather before departure — e.g. 08:45 for a 09:00 coach
Send 1-hour reminder? OPTIONAL
Yes / No — tick Yes for sessions delegates must not miss
Duration (minutes) OPTIONAL
Used for calendar exports; defaults to 60 min if not given
